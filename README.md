#  Car Evaluation Web Application

## Description
This project is a web application developed using PHP, MySQL, JavaScript, and HTML/CSS.  
It allows testers to evaluate car models and generates evaluation summaries.

---

## Features
- Tester registration
- Car evaluation system
- Form validation (JavaScript)
- Database storage (MySQL)
- Evaluation summary with averages

---

## Technologies
- PHP
- MySQL
- JavaScript
- HTML/CSS
- XAMPP

---

## How to Run

1. Install XAMPP  
2. Start Apache and MySQL  
3. Put project in:
   C:\xampp\htdocs\

4. Import the database:
   - Open http://localhost/phpmyadmin
   - Import the SQL file

5. Run:
   http://localhost//
